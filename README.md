## Setup Notes:

### Local Testing Steps

* Checkout gh-pages branch
* ```bundle update github-pages``` (Optional to keep dependencies up to date)
* ```bundle install```
* ```bundle exec jekyll serve```
* Goto 'http://localhost:4000/'
