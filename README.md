## Setup Notes:
### Local Testing Steps
* Checkout the gh-pages branch of this repo
* ```bundle update github-pages``` (Optional to keep dependencies up to date)
* ```bundle install```
* ```bundle exec jekyll serve```
* Goto 'http://localhost:4000/'
